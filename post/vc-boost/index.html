2. 使用

1) 头文件

拷贝C:\boost_1_79_0\boost文件夹到C:\boost

2) IDE设置

头文件引用

Project->Properties->Configuration Properties->VC++ Directories->General
Additional Include Directories: 增加 1）中根目录。

请参考VC19-header

库链接

Project->Properties->Configuration Properties->Linker->General
Additional Library Directories: 增加生成库目录

代码生成

vs2019 Project->Properties->Configuration Properties->c/c++->Code Generation
Spectre mitigation: 禁止

请参考VC19-C/C++

Example

#include <boost/thread.hpp>
#include <boost/chrono.hpp>
#include <iostream>

void wait(int seconds)
{
	boost::this_thread::sleep_for(boost::chrono::seconds{ seconds });
}

void thread()
{
	for (int i = 0; i < 5; ++i)
	{
		wait(1);
		std::cout << i << std::endl;
	}
}

int main()
{
	boost::thread t{ thread };
	t.join();
}
