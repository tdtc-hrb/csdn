0. 安装JDK

  • RHEL/CentOS
    参见 设置OpenJDK - CentOS
  • Ubuntu参见
    设置OpenJDK - Ubuntu

1. 安装Tomcat

下载并 解压

$wget https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-9/v9.0.63/bin/apache-tomcat-9.0.63.tar.gz
$tar -zxvf apache-tomcat-9.0.63.tar.gz

1) 移动文件夹

拷贝并重命名为tomcat

$sudo cp -R apache-tomcat-9.0.63 /usr/local/tomcat

2. fw

centos/rhel:

sudo firewall-cmd --permanent --zone=public --add-port=8080/tcp
sudo firewall-cmd --reload

centos5/6:

sudo iptables -A INPUT -p tcp --dport 8080 -j ACCEPT

3. 执行Tomcat

为了给tdtc(普通用户)有执行Tomcat的权利。

  • 使用超级用户
$su - root
  • 授权文件夹
$chown -R tdtc /usr/local/tomcat
  • 返回普通用户
$exit

1) 管理app

修改../conf/tomcat-users.xml
增加manager-gui角色，并授权给tomcat用户。

<role rolename="tomcat"/>
<role rolename="manager-gui"/>
<user username="tomcat" password="tomcat" roles="tomcat,manager-gui"/>

启动tomcat

$/usr/local/tomcat/bin/startup.sh
