设置固定IP

开发板访问NFS 服务器IP地址是固定的。

一、U16及以下

（Ubuntu16.04-）
使用root权限更改interfaces文件。

#gedit /etc/network/interfaces

完整例子：

/etc/network/interfaces

auto lo
iface lo inet loopback

auto eth0
iface eth0 inet static

address 192.168.0.111
netmask 255.255.255.0
gateway 192.168.0.1

二、U18及以上

（Ubuntu18.04+）

#vi /etc/netplan/50-cloud-init.yaml

network:
    ethernets:
        ens32:
            #dhcp4: true
            addresses:
            - 192.168.0.124/24
            gateway4: 192.168.0.1
            nameservers:
                addresses: [192.168.0.1, 219.146.3.226, 219.146.3.230, 219.146.3.238]
            optional: true
    version: 2

ens32：你的网卡，根据实际情况设置 dhcp4，gateway4：IPv4协议下的。

  • apply
#netplan apply

使用msmtp实现

installation

down

$wget https://marlam.de/msmtp/releases/msmtp-1.6.8.tar.xz

install

$xz -d msmtp-1.6.8.tar.xz
$tar xvf msmtp-1.6.8.tar
$cd msmtp-1.6.8
$./configure --prefix=/user/local/msmtp
$make

Then install using root

$make install

configuration

需要在user的根目录建立一个新文件：.msmtprc

$cd ~
$vi .msmtprc
$chmod 600 .msmtprc

.msmtprc

defaults

logfile ~/.msmtp/msmtp.log

# The SMTP server of the provider.
account love_shidie
host smtp.163.com
from love_shidie@163.com
auth login
user love_shidie@163.com
passwordeval gpg  --no-tty -q -d ~/.msmtp/love_shidie-password.gpg

# Set a default account
account default: love_shidie

r+w

$chmod 600 .msmtprc

SELinux的开启/关闭

(root权限)

SELinux(Security-Enhanced Linux) 是美国国家安全局（NSA）对于强制访问控制的实现，是 Linux历史上最杰出的新安全子系统。

1. 临时

临时的意思就是宽容模式(0)强制模式(1)与的切换 默认是：强制模式

  • 强制模式
$setenforce 1
  • 宽容模式 临时关闭
$setenforce 0

2. 永久

/etc/selinux/config

  • 开启
SELINUX=enforcing
  • 关闭
SELINUX=disabled

3. 查看SELinux的工作模式

getenforce

增加用户

ali云服务器

1. User Add and Set password

1) add user

useradd	site_user

2) set passwd

passwd -x 180 site_user

3) set home_dir

usermod -d /home/fdn site_user

2. assigning permissions

如果多个目录需要设置，则多次重复"(2)设置"即可。

文件系统是否支持acl

df -T | awk '{print $1,$2,$NF}' | grep "^/dev"
grep -i acl /boot/config*

分区是否支持acl

注意：在ali服务器上是vda1。

tune2fs -l /dev/vda1 | grep acl

1) 查看

目录的acl设置情况。

getfacl	/home/fdn

2) 设置

设置指定目录的acl。

setfacl -m user:site_user:rwx /home/fdn
