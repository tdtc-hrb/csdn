在CentOS安装Redis

Apr 11, 2020 #Redis #CentOS7 #Development Tools #wget #LimitNOFILE

0. preinstall

1) yum install

$yum groupinstall "Development Tools"
$yum install tcl wget

2) verification

$gcc -v
$make -v
$echo 'puts [info patchlevel];exit 0' | tclsh
  • User set
$ useradd redis
$ passwd redis

1. installation

1) install

$wget http://download.redis.io/releases/redis-4.0.14.tar.gz
$tar xzvf redis-4.0.14.tar.gz
$cd redis-4.0.14
$make
## Single Core
$taskset -c 0 make test
## sudo
$make install

2) config

$mkdir /etc/redis
$mkdir -p /var/redis
$cp /home/redis/redis-4.0.14/redis.conf /etc/redis

redis.conf

port  6379							         #default port is already 6379.
daemonize yes						         #run as a daemon
supervised systemd					     #signal systemd
pidfile /var/run/redis_6379.pid  #specify pid file
loglevel notice						       #server verbosity level
logfile /var/log/redis.log			 #log file name
dir  /var/redis/					       #redis directory

redis.service

$vi /etc/systemd/system/redis.service

[Unit]
Description=Redis In-Memory Data Store
After=network.target
[Service]
User=root
Group=root
ExecStart=/usr/local/bin/redis-server /etc/redis/redis.conf
ExecStop=/usr/local/bin/redis-cli shutdown
Restart=always
Type=Forking
LimitNOFILE=10032
[Install]
WantedBy=multi-user.target

2. command(redis service)

You do not need to execute the following commands when you are not maintaining it, only the first time.

$systemctl start redis
$systemctl enable redis
$systemctl status redis

3. test
