安装配置MySQL 5.7

Windows绿色安装
May 7, 2024

一、配置和安装

ini配置详见附录

预配置

需要vc12 redist

设置环境变量

名称：MYSQL57_HOME
赋值：D:\progFiles\mysql-5.7

init data

%MYSQL57_HOME%\bin\mysqld --initialize-insecure --console

注册/注销服务

注册

多个版本的MySQL运行：

%MYSQL57_HOME%\bin\mysqld --install MySQL57 --defaults-file="D:\progFiles\mysql-5.7\my.ini"

注销

sc delete mysql57

注意： 可以忽略如下警告信息：

CA certificate ca.pem is self signed.

Connection TLS Protocol Configuration

mysql> SHOW GLOBAL VARIABLES LIKE 'tls_version';

To change the value of tls_version, set it at server startup. For example, to permit connections that use the TLSv1.1 or TLSv1.2 protocol, but prohibit connections that use the less-secure TLSv1 protocol, use these lines in the server my.cnf(my.ini - windows) file:

# only support v1.2
tls_version=TLSv1.2

configuration CA

# enable the server for encrypted connections
ssl_ca=ca.pem
ssl_cert=server-cert.pem
ssl_key=server-key.pem

二、使用mysqladmin和mysql

1．修改密码

格式：mysqladmin -u用户名 -p旧密码 password 新密码

1) 空密码

注：如果root没有密码，-p旧密码, 可以省略

%MYSQL57_HOME%\bin\mysqladmin -uroot password qazxsw

2) 有密码

%MYSQL57_HOME%\bin\mysqladmin -uroot -pqazxsw password mypassword123456

2. 连接mysql

格式： mysql -h主机地址 -u用户名 －p用户密码

1) 本机

%MYSQL57_HOME%\bin\mysql -uroot -p

2) 远程主机

%MYSQL57_HOME%\bin\mysql -h192.168.2.101 -uroot -pqazxsw

附：my.ini

[mysqld]
innodb_buffer_pool_size = 1012M

# These are commonly set, remove the # and set as required.
basedir = D:/progFiles/mysql-5.7/
datadir = D:/progFiles/mysql-5.7/data
port = 3306

character-set-server=utf8mb4
# only support v1.2
tls_version=TLSv1.2
[client]
port=3306
plugin-dir=D:/progFiles/mysql-5.7/lib/plugin
