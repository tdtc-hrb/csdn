The GNU Privacy Guard

Note: normal user login.

1. install

$yum install gnupg

$yum install rng-tools

2. gen key

$gpg --gen-key

Press Key: O 输入完，按“Tab”切换到“OK”,按回车。 再输入一次，

3. gen revoke

$gpg gen-revoke 7209DBAD

Press Key: y

4. encrypt file(password)

1) password-file

$vi love_shidie-password

密码文件(明文密码)

thisispassword

2) gen gpg file

Gen file: love_shidie-password.gpg

$gpg -e -r veic_2005@163.com love_shidie-password
