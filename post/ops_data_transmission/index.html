Linux与Windows互传数据

Bootstrap3
May 15, 2020

一、光盘方式

VMware

1. 制作ISO

打开UltralISO，把需要的文件拖拽过去即可。

2. Linux

1) look

ls -l /dev | grep cdrom

2) load

sudo mount /dev/cdrom /mnt/

3) unload

sudo umount /mnt

二、scp

1. Windows->Linux

Windows的scp包含在putty工具中。

pscp f:\docker-soft\docker-compose-Linux-x86_64 tdtc@192.168.211.131:/home/tdtc/

pscp f:\docker-soft\protobuf-cpp-3.11.4.tar.gz tdtc@192.168.211.131:/home/tdtc/

pscp f:\docker-soft\go1.13.8.linux-amd64.tar.gz tdtc@192.168.211.131:/home/tdtc

2. Linux->Windows

注意：仅适用于win10.

1）增加OpenSSH Server

安装OpenSSH

开始菜单-》设置-》应用-》管理可选功能:

开启OpenSSH

注意：关闭Windows防火墙或允许22端口。

2）传送文件

在Linux下操作。

scp /home/tdtc/go1.13.8.linux-amd64.tar.gz xiaobin@192.168.211.112:/D:\war
