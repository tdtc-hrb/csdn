使用go-bindata

Apr 10, 2022

安装

$ go get -u github.com/go-bindata/go-bindata/...

generation bindata.go

$ cd ~/go/src/github.com/xiaobin80/go-micro-services/
$ go-bindata data/...
$ mv bindata.go ./data/

修改包名

package main

—》

package xxx

此例xxx为data。
