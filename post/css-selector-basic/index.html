Selector type CSS 说明
Tag name p {} 对整个网页中标签有效
ID #some-id {} 对网页中指定id有效
Class .some-class {} 对网页中相同类名有效

元素

document的元素就是最基本的选择器。

html {color:black;}
h1 {color:blue;}
h2 {color:silver;}
