CentOS修改网卡为固定IP

需要net-tools支持
Mar 13, 2022

View

$ip addr

1 is loopback
2 is the current network card

Setup

auto

Start job network adapter.

ONBOOT=yes

static

cd /etc/sysconfig/network-scripts
$sudo vi ifcfg-*

*: current network card

BOOTPROTO=static
IPADDR=192.168.1.186
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=211.161.191.235
DNS2=211.161.122.197

virtual machine

vmware

设置Network Adapter为Bridged

缺省/etc/sysconfig/network-scripts/ifcfg-eth0

default: eth0

vbox

设置Network Adapter为Bridged

缺省/etc/sysconfig/network-scripts/ifcfg-enp0XX

default: enp0s3
