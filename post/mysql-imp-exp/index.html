MySQL导入导出数据库

Sep 10, 2022

导出数据库

$mysqldump dbName > dbName.sql -u user -p
  • v5.6/5.7
$mysqldump --no-defaults dbName > dbName.sql -u user -p

导入数据库

mysql>CREATE DATABASE `dbName` CHARACTER SET utf8 COLLATE utf8_general_ci;

mysql>USE dbName;
